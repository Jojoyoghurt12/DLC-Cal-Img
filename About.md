### Jesse ten Broeke - Msc Biomedical Sciences - *Neuroscience*

---
*This repository serves as a pipeline for efficiently integrating and comparing behavioural data to calcium imaging data by matching location through time to activity through time.*

> [!Note] 
> This repository is open-sourced and can be used by whoever seems fit. 

---
> The code in this repository uses DeepLabCut acquired data as the standard location input, however any program that provides location on a frame-by-frame basis can be used. 
For more information regarding the usage of the software visit their [Official GitHub Page](https://github.com/DeepLabCut/DeepLabCut/blob/main/README.md).
